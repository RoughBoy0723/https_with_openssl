한밭대학교 4학년 캡스톤 디자인 프로젝트입니다.

openssl을 활용한 https web server을 라즈베리파이에서 동작하는게 최종 목표이며

3명의 학생(김범진, 오원영, 민평화)들이 캡스톤을 진행합니다.

3단계로 나누어서 진행하였습니다.

1. socket를 사용하여 web server 구현

2. openssl을 사용하여 암호화된 통신(https)을 하는 web server 구현

3. Raspberry Pi에 이식

이 과정을 C언어를 사용하여 진행하였고 그와 동시에

소켓의 작동방식과 http, openssl의 암호화 방식과 암호화된 정보를 교환하는 방식에 대하여 이해하었고 구현할 능력을 지니게 되었습니다. 

This is a 4th year capstone design project at Hanbat University.

The final goal is to implement an https web server.

Three students will conduct the capstone.
